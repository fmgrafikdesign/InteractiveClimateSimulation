Project title: Interactive Climate Simulation
Project members: Fabian Mohr, Matthias Köttritsch
Supervisor: Martin Harrer
Abstract:
In this project, a browser-based interactive terrain simulation was created where the user can
influence climate and geographical conditions like temperature, humidity, etc. by turning a few physical wheels. 
The terrain is generated from an user-selected section of a map. The terrain then simulates how
the changed conditions play out, so you see the effect of those base parameters in real time, 
offering a playful approach to learn a lot about the effects climate and weather has on terrain. 
This also makes the project suitable for multiple users, ie. in an exhibition or museum.

In diesem Projekt wurde eine im Brwoser laufende, interaktive Terrainsimulation realisiert, bei der der Benutzer klimatische und
geografische parameter wie Temperatur, Feuchtigkeit etc. durch das drehen einiger Stellräder verändern kann. Das Terrain wird zuerst 
bestimmt durch einen vom Benutzer ausgewählten Kartenausschnitt und dann simuliert, wie sich die Veränderung der Parameter auswirkt.
Dadurch kann man die Veränderung in Echtzeit mitverfolgen, was einen spielerischen Ansatz an Klima- und Terrainveränderung bietet, 
was die Anwendung gut für mehrere Benutzer z.B. in einem Museumsumfeld macht.
